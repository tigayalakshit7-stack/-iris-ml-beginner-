\# Iris ML Beginner Project  



A simple Machine Learning project for absolute beginners using the Iris dataset. 



\## Features 

\- Loads Iris dataset from scikit-learn 

\- Trains a RandomForest model 

\- Evaluates accuracy 

\- Saves model as `.pkl` 

\- Predicts output from saved model 



\## Setup 



pip install -r requirements.txt 



Train 

python train.py 



Predict 

python predict.py
